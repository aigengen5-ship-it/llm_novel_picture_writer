# llm_novel_picture_writer

이 프로그램은 기본적인 라노벨(초기 설정: 6개의 에피소드) 제작 및 이에 동기화된 이미지를 제작합니다. openAPI를 사용하며 anima prompt를 생성, 또는 comfyui를 직접 부를 수 있습니다.

*   ComfyUI(Not necessary).

---------------------

**Usage:**

1.  `git clone https://github.com/aigengen5-ship-it/llm_novel_picture_writer`
2.  Create a virtual environment: python3 -m venv ./venv
3.  Activate the virtual environment: `source venv/bin/activate`
4.  Install dependencies: `pip install -r requirement.txt`
5.  Run the script: run_short_epi6.scr(or you can add your dialog)

**Output:**

*   Story text will be saved in the `result` folder.
*   Generated images will be located in your ComfyUI output directory.

---------------------

**Configuration:**

If you want to increase the episode number, modify two files as follows.

*  please modify plot_json.txt and update episode_num.
*  please modify ./story/SHORT/plot_short_epi6.txt; add or remove this tag same as episode num.

[EPISODE_GEN:6]

---------------------

**Write your own short novel:**

If you want to write your novel, please follow these steps. 
a) Please update ./story/SHORT/order.txt for your novel writing.
  * Do not remove ##SPLIT##, it is internallly used. 
  * There are four sentences splitted by ##SPLIT##
  1) You can change Character Sheet, please keep the format or you can modify python code directly.
  2) You can change Story Override, it will control the whole novel style.
  3) You can change detailed story/episode section, more detailed, more episode number will be nedeed.
  4) You can change the background, default background is for the personal usage. Please modify it if necessary.
